# Translator Chat

Korean-Japanese Translation Chat Service

## 프로젝트 소개
실시간 번역 채팅 서비스

## 기능 요구사항

1. Discord 스타일 UI 구현
2. 2명 이상 실시간 채팅 가능
3. 사용자 닉네임 표시
4. 사용자 메시지는 왼쪽 정렬
5. 번역봇 메시지는 오른쪽 정렬
6. 사용자가 입력한 모든 메시지 자동 번역
7. 번역 결과를 채팅창에 출력
8. 채팅 기록 DB 저장 (선택)
9. 재접속 시 이전 채팅 조회 (선택)
10. 한국어 ↔ 일본어 번역 지원

## 비기능 요구사항

1. 실시간 메시지 전송
2. 반응형 웹 지원
3. Discord와 유사한 UI 구성
4. 사용자 식별 가능
